## 📌 About the Project

Zephyr is a modern fashion e-commerce website designed and developed as part of an internship project. It features a complete online shopping experience — from browsing products by category to placing an order — built entirely with frontend technologies.

> *"Step into ZEPHYR, where modern sophistication meets effortless style."*

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and content |
| CSS3 | Custom styling and animations |
| Bootstrap 4.5 | Responsive layout and components |
| jQuery | Form validation and interactivity |
| Font Awesome 5 | Icons throughout the site |

---

## 📄 Pages

| Page | File | Description |
|---|---|---|
| Home | `index.html` | Hero slider, categories, latest products, features |
| Products | `product.html` | Full product listing with category & price filters |
| Product Detail | `product_detail.html` | Image carousel, description, add to cart |
| Category | `category.html` | Browse products by category |
| Cart | `cart.html` | Cart items with quantity controls and order summary |
| Checkout | `checkout.html` | Billing form and order summary |
| Order Confirmation | `order_confirmation.html` | Success message with order details |
| About Us | `about.html` | Brand story, testimonials, features |
| Contact Us | `contact.html` | Contact info and validated contact form |
| Search | `search.html` | Product search results listing |

## ✨ Features

- ✅ Fully responsive design — works on mobile, tablet, and desktop
- ✅ Hero image slider / carousel on homepage
- ✅ Shop by Category section with 6 categories
- ✅ Product listing page with category filter and price range slider
- ✅ Individual product detail pages with image carousel and related products
- ✅ Add to Cart with quantity increase / decrease controls

- ✅ Contact form with jQuery & JavaScript validation
- ✅ Testimonials carousel on About Us page
- ✅ Scroll to top button on every page
